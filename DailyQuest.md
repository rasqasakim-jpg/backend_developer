
## 1. Apa itu Node.js dan untuk apa biasanya digunakan?

Node.js itu sebenarnya lingkungan runtime yang bikin JavaScript bisa jalan di luar browser. Biasanya dipakai buat bangun aplikasi backend, bikin API, ngerjain proses yang butuh real-time kayak chat atau notifikasi, sampai tugas-tugas server lainnya. Karena Node.js itu non-blocking dan cepat, banyak banget dipakai buat aplikasi yang butuh performa tinggi.

## 2. Perbedaan Node.js dan JavaScript di Browser

Di browser, JavaScript fokusnya buat ngatur tampilan dan interaksi user: klik tombol, animasi, manipulasi DOM, dan sebagainya. Sementara kalau di Node.js, JavaScript dipakai buat ngurus server-side logic: akses database, handle request-response, sampai ngatur file system. Browser punya fitur bawaan kaya DOM dan window, sedangkan Node.js punya module bawaan seperti fs, http, dan path.

## 3. Fungsi Module Import di Node.js

Module import itu dipakai buat nge-load fungsi, variabel, atau logic dari file lain supaya kode lebih rapi dan gampang dikelola. Jadi kita nggak perlu nulis semuanya di satu file. Dengan import, kita bisa misahin fitur sesuai kebutuhan dan tinggal manggil modulnya saat diperlukan.
